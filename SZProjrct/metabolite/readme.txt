Metabolite analysis

we got 171 samples : 81*90 (control&case)

According to the following principles, we selected those metabolites
(1)以一级电离score和二级电离fragment score排序筛选， 重复的离子挑选唯一值
(2)retention_time: 0.5-9分钟； peak width 1-30s, ppm质量误差绝对值 < 10
(3)标准化：对每一个样本做normalize

we did PERMANOVA analysis && construct RF classifier
